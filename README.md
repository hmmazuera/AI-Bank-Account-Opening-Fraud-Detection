# AI Bank Account Opening Fraud Detection

End-to-end fraud detection project for bank account opening applications using Machine Learning and Deep Learning.

## Project Overview

This project predicts the risk of fraud in bank account opening applications.

The objective is to build an internal fraud risk tool for banking and fintech teams, comparing traditional Machine Learning models with Deep Learning models.

## Problem Statement

Banks and fintech companies need to identify potentially fraudulent account opening applications before approval.

This project uses structured applicant and application data to classify applications as legitimate or potentially fraudulent.

## Dataset

Dataset: Bank Account Fraud Dataset  
Source: Feedzai / NeurIPS 2022

## Models

The project compares:

- Logistic Regression
- Random Forest
- XGBoost / LightGBM
- Deep Neural Network
- TabNet or Deep Learning baseline
